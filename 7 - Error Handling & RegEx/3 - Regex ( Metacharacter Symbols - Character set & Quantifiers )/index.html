Metacharacter Symbols

    Must start with

  • /^h/i

    • Must ends with

  • /d$/i

    • Must begin and end with

  • /^hello$/

    • Matches any ONE character

  • /^h.llo$/

    • Matches any character 0 or more times

  • /^h*llo$/

    • Optional character

  • /gre?a?y/i

    • Escape character

  • /gre?a?y\?/i

Character set & Quantifiers

    Must be an a or e

  • /gr[ae]y/i

    • Must be a G or F

  • /[GF]ray/i

    • Must begin a G or F

  • /^[GF]ray/i

    • Must anything except a G or F

  • /[^GF]ray/i

    • Match any uppercase letter

  • /[^A-Z]ray/

    • Match any lowercase letter

  • /[^a-z]ray/

    • Match any letter

  • /[^A-Za-z]ray/

    • Match any digit

  • /[0-9]ray/

Breces - {} Quantifiers

    Must occur exacly {m} amount of times

  • /hel{2}o/i

    • Must occur 2 to 4 {m} amount of times

  • /hel{2,4}o/i

    • Must occur at least {m} times

  • /hel{2,}o/i

Parentheses () - Grouping

  • /^([0-9]x){3}$/

Short Character Classes

    Word character - alphanumeric or _

  • /\w/

    • + = one or more

  • /\w+/

    • Non-Word character

  • /\W/

    • Match any digit

  • /\d/

    • Match any digit 0 or more times

  • /\d+/

    • Match any non-digit

  • /\D/

    • Match whitespace char

  • /\s/

    • Match non-whitespace char

  • /\S/

    • word boundary

  • /Hell\b/i

Assertions

    Match x only if followed by y

  • /x(?=y)/

    • Match x only if NOT followed by y

  • /x(?!y)/
